# Contributing

Project Genesis follows a research-first and user-first philosophy. Contributions should support thoughtful decisions, calm technology, and long-term value for users.

Contributors should:

- Understand the project vision before proposing changes.
- Make decisions based on evidence instead of assumptions.
- Prefer simplicity over unnecessary complexity.
- Prefer solving real user problems over adding more features.
- Keep documentation updated.
- Write meaningful commit messages.
- Submit focused pull requests.
- Respect user privacy and well-being in every proposal.
- Discuss important architectural decisions before implementation.

Thank you for helping build calm, thoughtful technology.
